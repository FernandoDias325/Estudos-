## 👨‍🎓 Sobre o Repositório

Este repositório centraliza todo o conteúdo prático da minha jornada como estudante de **Engenharia de Software**, reunindo exercícios, projetos e experimentos desenvolvidos nos cursos da **Alura**.

O objetivo é simples: documentar minha evolução, manter um histórico de aprendizado organizado e criar uma base de referência para consultas futuras.

## 🎯 Objetivos

- Consolidar conceitos teóricos através da prática
- Manter um histórico organizado da evolução técnica
- Revisitar e refatorar código ao longo do tempo
- Experimentar novas tecnologias com segurança
- Construir uma base sólida para o mercado de trabalho
